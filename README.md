# Car-Price-Prediction
Car Price Prediction is a machine learning project that estimates the price of used cars based on features like brand. It uses Lasso and Ridge Regression to improve accuracy and prevent overfitting through regularization techniques. The model is evaluated using metrics like MAE, MSE, and R² score to identify the best-performing approach.


🚗 Car Price Prediction using Lasso & Ridge Regression

This project focuses on building a machine learning model to predict the price of used cars based on various features such as brand, model, year, fuel type, transmission, mileage, and more. The goal is to help users and businesses estimate a fair market value for vehicles using data-driven insights.

The dataset is first preprocessed by handling missing values, encoding categorical variables, and performing feature scaling. Exploratory Data Analysis (EDA) is conducted to understand relationships between features and the target variable (car price).

To improve model performance and prevent overfitting, two regularization techniques are applied: Lasso Regression (L1 regularization) and Ridge Regression (L2 regularization). These models help in selecting important features and stabilizing predictions by penalizing large coefficients.

The performance of both models is evaluated using metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and R² score. A comparison is made to determine which model provides better accuracy and generalization.

This project demonstrates practical implementation of regression techniques, feature engineering, and model optimization, making it useful for real-world applications in the automobile and resale industry.
